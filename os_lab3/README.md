# Лабораторная работа 3

Программа для взаимодействия между процессами через разделяемую память и сигналы.

## Сборка

```bash
make
```

## Запуск

```bash
./parent
```

Программа попросит ввести имя файла для результата и числа через пробел. Например:

```
Enter result file name: output.txt
Enter numbers (separator - space): 12.5 2 0.5 5
```

Результат будет записан в указанный файл.

## Как это работает

Родительский процесс создаёт разделяемую память, записывает туда числа и запускает дочерний процесс. Дочерний процесс читает числа из разделяемой памяти, делит первое число на остальные и записывает результат в файл. Синхронизация происходит через сигналы SIGUSR1 и SIGUSR2.

## Файлы

- `parent.c` - родительский процесс
- `child.c` - дочерний процесс  
- `common.h` - общие определения
- `Makefile` - сборка

## Очистка

```bash
make clean
```
